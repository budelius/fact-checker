<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Untrusted transcript/metadata rendered as HTML | high | Components render transcript, OCR/source-clue text, and metadata as text nodes only; do not use `dangerouslySetInnerHTML`. |
| UI implies truth evaluation exists | high | Claim rows show `evidence_status: pending` only and avoid supported/contradicted/true/false labels. |
| Progress is inaccessible or invisible | medium | Use `aria-live="polite"` for progress and `role="alert"` for blocking recoverable errors. |
| Long URLs/UUIDs break mobile layout | medium | Use `overflow-wrap: anywhere`, fixed thumbnail dimensions, and 44px mobile touch targets. |
| Duplicate submit creates confusing jobs | low | Disable duplicate submit while a request for the same URL or selected video is pending. |
</threat_model>
