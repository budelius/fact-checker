## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| UI invents or transforms truth labels | high | Type report labels from API and render them only. |
| User cannot run report because Phase 3 has no UI | high | Add a small ground-truth API client and let `Generate report` start ground truth first when needed. |
| Long citations/UUIDs/URLs break layout | medium | Add wrapping rules and stable row dimensions. |
| UI implies creator rating or truth score | high | Use claim-level labels only and avoid rating/truth score copy. |
| Provenance rail is disconnected from selected report | medium | Lift selected report metadata to `AppShell` or pass a right-rail render prop/state. |
</threat_model>
