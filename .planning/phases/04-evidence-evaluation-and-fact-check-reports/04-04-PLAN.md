<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| API starts from missing or incomplete ground-truth jobs | high | Return typed 404/400 errors and do not run evaluator without claims. |
| Pipeline persists model output before validation | high | Validation stage must run before Markdown/Mongo writes. |
| Report route conflicts with `/reports/jobs/...` path matching | medium | Define fixed `/jobs/...` routes before `/{report_uuid}`. |
| E2E tests accidentally call live OpenAI/Mongo/Qdrant | high | Use dependency overrides, fake evaluator, fake repositories, and temporary vault roots. |
| Failed jobs hide trace data | medium | Preserve job UUID, current operation, failed stage, and validation errors. |
</threat_model>
