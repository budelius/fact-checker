<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Rerun overwrites a prior report | high | Version helper always creates a new report UUID/path/version. |
| Mixed labels create misleading `supports` or `contradicts` graph edges | high | Use `cites` plus provenance for mixed; only create support/contradict edges when label semantics are clear. |
| Insufficient claims appear as scientific conclusions | high | Markdown explicitly states missing direct evidence and writes no support/contradict edge. |
| Provenance appendix omits unused candidates | medium | Markdown builder requires cited and candidate evidence sections. |
| Report paths collide | medium | Slug includes video or ground-truth job identity and version. |
</threat_model>
