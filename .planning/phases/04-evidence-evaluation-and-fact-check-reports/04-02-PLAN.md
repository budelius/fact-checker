<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Paper summaries are cited as verdict evidence | high | Evidence selector excludes summaries; validator rejects summary-like citation kinds. |
| Model produces an uncited supported/contradicted/mixed label | high | Validation rejects non-insufficient labels without valid candidate citations. |
| Model cites evidence outside the retrieval set | high | Validation checks evidence UUID and chunk ID against per-claim candidates. |
| Prompt injection from transcripts or papers alters instructions | high | Wrap all source text with `wrap_untrusted_text` and keep instructions outside untrusted blocks. |
| Adjacent evidence creates false support | high | Prompt requires direct evidence; no direct evidence maps to `insufficient`. |
</threat_model>
