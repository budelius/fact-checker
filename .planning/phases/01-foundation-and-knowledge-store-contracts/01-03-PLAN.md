<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Markdown notes contain secrets | high | Vault schema explicitly bans API keys, tokens, passwords, and credentials in frontmatter/body. |
| Raw evidence accidentally mutated | medium | `vault/SCHEMA.md` marks `vault/raw/` as immutable evidence input. |
| Broken identity from slug-only filenames | medium | Templates require `uuid` frontmatter and document slug collision handling. |
| Prompt injection preserved without boundary | medium | Schema labels raw transcripts, papers, web pages, captions, and comments as untrusted content. |
</threat_model>
