## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Generated Markdown provenance is corrupted by browser edits | high | Only add annotation records through a separate collection/service; never write annotation text into parsed canonical note content. |
| Later plans create incompatible payload shapes | high | Centralize browse/search/graph/consistency/rating summary schemas in `backend/app/schemas/knowledge.py`. |
| Parallel backend plans conflict on route registration | medium | Register `/knowledge` and placeholder feature subrouters in this plan; later plans modify only their owned subrouter files. |
| Consistency checks cannot inspect Qdrant payloads | medium | Add `scroll_payloads` to `QdrantRepository` now so graph/consistency can read payload metadata without editing Qdrant later. |
</threat_model>
