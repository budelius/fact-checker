<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Ratings become punitive reputation scores | high | Use only approved evidence-state labels and add tests forbidding trust/reputation/truth score copy. |
| Rating confidence is opaque | high | Return evidence count, label distribution, source basis, cited report versions, and relationship basis in every record. |
| Early ratings overstate confidence | high | Mark records experimental until `evidence_count >= 10`. |
| Rating Markdown is stored outside the vault contract | medium | Add `vault/wiki/ratings/` to `vault/SCHEMA.md` and write generated records there. |
</threat_model>
