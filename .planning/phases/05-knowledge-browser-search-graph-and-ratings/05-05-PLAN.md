<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| UI presents a landing page instead of the actual browser | high | Keep `AppShell` as the first screen and default to a selected knowledge page or clear empty browser state. |
| Frontend computes ratings or graph relationships independently | high | Fetch knowledge, graph, consistency, search, and rating payloads through `frontend/src/api/knowledge.ts`. |
| Rating copy becomes punitive | high | Use approved badge labels only and add grep verification for forbidden phrases. |
| Long IDs/URLs break layout | medium | Add `overflow-wrap: anywhere` to metadata, search snippets, graph labels, and path/UUID elements. |
| Graph view becomes inaccessible on mobile | medium | Provide a relationship-list fallback and stack right rail below content on narrow widths. |
</threat_model>
